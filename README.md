# Reaction Time Bot

A simple Python bot that automates the [Human Benchmark Reaction Time Test](https://humanbenchmark.com/tests/reactiontime) with a simple gui
## Features

- Detects the green screen using pixel color
- Automatically clicks to record reaction time and restart
- Start/Stop control via GUI
- Lightweight and beginner-friendly

## GUI Preview

The app includes a minimal GUI with:
- Start and Stop buttons
- Status indicator

## Requirements

- Python 3.7+
- `pyautogui`
