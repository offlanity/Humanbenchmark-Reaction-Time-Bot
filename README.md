# Reaction Time Bot

A simple Python bot that automates the [Human Benchmark Reaction Time Test](https://humanbenchmark.com/tests/reactiontime) with a simple gui

## GUI Preview
 ![Image Alt](https://github.com/offlanity/Humanbenchmark-Reaction-Time-Bot/blob/main/preview.png?raw=true)

- Start and Stop buttons
- Status indicator

## Requirements

- Python 3.7+
- `pyautogui`
- `tkinter`
